# Portfolio de Eliezer Ponce

Portfolio personal construido con React, Bootstrap y GitHub Pages.

## Contenido

- Presentación profesional enfocada en React, React Native, .NET y SQL Server.
- Secciones de proyectos, stack técnico, herramientas y CV.
- Caso destacado de Sistema Ink Multiservicios como sistema de gestión cliente-servidor.
- Rutas compatibles con GitHub Pages mediante `HashRouter`.

## Scripts

### `npm start`

Ejecuta el proyecto en modo desarrollo.

### `npm run build`

Genera la versión optimizada para producción.
